### Security & Encryption
- **[ColumnEncryptor](https://github.com/Tricksfor-Organization/ColumnEncryptor)** - Encrypt database column values in Entity Framework using AES-GCM per-row encryption with key versioning and ASP.NET Core Data Protection integration.

### Logging & Observability
- **[LogManager](https://github.com/Tricksfor-Organization/LogManager)** - Serilog-based logging library for .NET with support for Console, File, Elasticsearch, and Grafana Loki sinks, plus correlation ID tracking and production-ready enrichers.

### Blockchain Integration
- **[BlockChainTools](https://github.com/Tricksfor-Organization/BlockChainTools)** - Comprehensive toolkit for blockchain interactions including transactions, queries, gas estimation, Web3 providers, and related services.
- **[DistributedNonce](https://github.com/Tricksfor-Organization/DistributedNonce)** - Blockchain distributed nonce service designed for scalable environments to manage transaction ordering.

### Distributed Systems
- **[DistributedLockManager](https://github.com/Tricksfor-Organization/DistributedLockManager)** - Handle execution of tasks or functions with distributed locks for use in scalable, multi-instance applications.

### Utilities & Tools
- **[CryptoFairPicker](https://github.com/Tricksfor-Organization/CryptoFairPicker)** - Fair, unbiased, and cryptographically secure winner selection library for games, lotteries, and other scenarios where fairness is critical. Built with Dependency Injection support.
